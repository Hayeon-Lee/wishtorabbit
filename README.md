### 프로젝트 설명

한 달에 한 번씩 이루고 싶은 목표를 달 토끼에게 말해요.

그리고 자신의 달을 친구들에게 공유하세요! 친구들이 여러분의 달에 놀러와 응원을 남겨줄거예요.

그렇게 남긴 응원들은 하나씩 재료가 되어 한 달이 끝날 때 떡이 되어있답니다 🥰

1년동안 열심히 목표를 이뤄 하나의 떡 박스를 만들어보아요!

(이 버전은 친구들에게 공유가 되지 않습니다)
